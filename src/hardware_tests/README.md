# ABOUT

The tables below describe how the various terminals are controlled. Use the test files to populate them and then use the tables to populate the `config.json` file.

# SENSORS

| Terminal | Address | Channel | GPIO Pin |
|----------|---------|---------|----------|
| Left-1   | 0x48    | 0       | 18       |
| Left-2   | 0x48    | 1       | 19       |
| Left-3   | 0x48    | 2       | 20       |
| Left-4   | 0x48    | 3       | 21       |
| Left-5   | 0x49    | 0       | 22       |
| Right-1  | 0x49    | 1       | 28       |
| Right-2  | 0x49    | 2       | 26       |
| Right-3  | 0x49    | 3       | 27       |
